Projeto desenvolvido com acompanhamento de aulas práticas da plataforma Rocketseat.

## Funcionalidades

*   Desenvolvimento estruturado com boas práticas de semântica em HTML e estilização em CSS.
*   Uso estratégico de animações para melhorar a experiência visual do usuário.
*   Rolagem de página fluida e navegação limpa.

## Tecnologias Utilizadas

*   **Front-end:** HTML e CSS
